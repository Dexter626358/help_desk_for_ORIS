Формирование csv файла выпуска для добавления пдф статей к опубликованному выпуску

Инструкция

  1. Укажите ссылку на опубликованный выпуск (например, https://journals.rcsi.science/2713-3192/issue/view/23830).
  2. Загрузите ZIP архив с PDF файлами статей.
  3. Сервис найдет статьи выпуска, возьмет только статьи с DOI и сформирует CSV без заголовков.
Формат CSV: doi;имя_pdf_файла;en_US
Разделитель: ;, заголовки не добавляются.
Поддерживаются только файлы .zip
