# XSD Схемы для валидации XML

В этой директории размещаются XSD схемы для валидации XML файлов.

## Использование

1. Поместите вашу XSD схему в эту директорию
2. При валидации XML файла выберите нужную схему из списка
3. Система проверит XML файл на соответствие выбранной схеме

## Пример схемы

В директории уже есть пример схемы `journal.xsd` для валидации XML файлов журналов.

## Формат имени файла

Схемы должны иметь расширение `.xsd` для автоматического обнаружения системой.
